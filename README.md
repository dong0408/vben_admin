## 特性

- **技术栈**：使用 Vue3/vite 等技术开发
- **主题**：提供多套主题色彩，可配置自定义主题
- **国际化**：内置完善的国际化方案
- **权限**：内置完善的动态路由权限生成方案

## 安装使用

1. 安装依赖

```bash
pnpm install
```

2. 运行

```bash
pnpm dev
```

3. 打包

```bash
pnpm build
```
